## 四、推广奖励权益

### 4.1 直推奖励（一次性佣金）

推荐他人购买节点，**立即结算一次性佣金**，按被推荐人投资额计算。

| 被推荐节点 | 直推奖励比例 | 示例（推荐一席符主 L4 · 50,000 U） |
| --- | --- | --- |
| 符胚 L1 · PIONEER | **5%** | 获得 125 U |
| 符印 L2 · BUILDER | **8%** | 获得 400 U |
| 符魂 L3 · GUARDIAN | **10%** | 获得 1,000 U |
| 符主 L4 · STRATEGIC | **15%** | 获得 7,500 U |

### 4.2 创世节点（L5 · 条件触发）

创世节点 **非购买获得**，通过以下**任一条件**自动达成：

- **条件一：** 直推 ≥ 5 个超级节点（符主 · L4）。
- **条件二：** 团队矩阵内累计 ≥ 10 个超级节点（符主 · L4）。

**创世权益：**

1. 正常参与节点权重分红（保留已购节点等级的权重）。
2. **额外**从 **核心激励池 10%** 中按创世权重比例分配：

```
genesisPool         = coreIncentivePool × 10%
userGenesisReward   = (userWeight / genesisTotalWeight) × genesisPool
```

## 七、核心计算公式

```
获得母 TOKEN 数量 = 投资额(U) ÷ 私募价($/枚)
开盘浮盈($)     = 获母 TOKEN 数 × (开盘价 $0.028 - 私募价)
持仓市值(U)     = 获母 TOKEN 数 × 当前价格
日收益率(%)     = 日 USDT 收益 ÷ 投资额
30 天累计 USDT  = 日 USDT 收益 × 30
空投价值(U)     = 空投总量(枚) × 当前价格
全网权重占比(%) = (本节点权重 × 席位数) ÷ 全网权重总积分(1,680)
直推佣金(U)     = 被推荐人投资额 × 直推奖励比例
创世奖励(U)     = (本节点权重 / 创世节点总权重) × (核心激励池 × 10%)
```

## 八、关键系统逻辑（给 coder）

### 用户状态

每个用户维护：
- 当前节点等级（L1 / L2 / L3 / L4）
- 是否达成创世（boolean，自动判定）
- 推荐人
- 团队结构（用于统计超级节点数量）

### 节点购买流程

1. 扣款
2. 发放母 TOKEN（锁仓，按阶段解锁）
3. 触发一次性直推奖励

### 空投释放

按全局参数（TLP 底池 / 团队有效质押 / 上线天数）触发阶段解锁比例。

### 创世判定

```ts
if (directL4Count >= 5 || teamL4Count >= 10) {
  user.isGenesis = true;
}
```

### 分红执行

```
totalWeight  = Σ (userWeight_i)
userReward   = (userWeight / totalWeight) × dividendPool
genesisPool  = coreIncentivePool × 10%
userExtra    = (userWeight / genesisTotalWeight) × genesisPool  // Genesis 用户
```
