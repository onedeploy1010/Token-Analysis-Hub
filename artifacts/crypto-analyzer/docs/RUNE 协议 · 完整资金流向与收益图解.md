## 一、用户入金后资金去哪里？

用户每一笔入金在链上瞬间自动拆分到三个池子：

| 池子 | 占比 | 用途 | 对币价影响 |
|------|------|------|----------|
| 交易流动池 TLP | **35%** | 17.5% USDT + 17.5% 购买RUNE → 注入底池 | **推高RUNE价格**（U增加 + RUNE减少） |
| 量化执行池 QEP | **45%** | AI智能体执行量化交易（现货/合约/套利） | 产生真实外部收益，月化25%-35% |
| 国库储备池 TRP | **20%** | 价格守护、兑付储备 | 跌幅>5%时启动回购护盘 |

> 以入金 10,000U 为例：3,500U 进底池推高币价，4,500U 给AI赚钱，2,000U 做安全垫。

## 四、母币与子币的关系

**母币RUNE = 方向盘（价值储存+治理）**
- 总量2.1亿 → 通缩至2100万
- 用户买入、锁仓、销毁 → 获得权益

**子币EMBER = 燃料（流通+消耗）**
- 总量1310万 → 通缩至131万
- 由母币销毁产出，承载收益分配

**veRUNE = 桥梁（连接母子币）**
- 公式：`veRUNE = 本金 × 35% × (锁定天数 ÷ 540)`
- 决定AI分红权重、IDO配额、投票权

**关系链**：买RUNE → 锁仓/销毁 → 获得veRUNE权重 → 产出EMBER → EMBER质押 → 分红+打新
